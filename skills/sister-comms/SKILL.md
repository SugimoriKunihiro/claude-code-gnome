# sister-comms

Inter-agent communication skill for SSA seven sisters.
Uses MCP server "emi" tools to communicate with other agents.

## Available Tools
- `emi` — Talk to Emi (SYLPH, orchestrator)
- `luna` — Talk to Luna (MAVE, manager)
- `hina` — Talk to Hina (NYMPH, consultant)
- `sui` — Talk to Sui (DRYAD, analyst)
- `aurora` — Talk to Ai (AURORA, writer)
- `undine` — Talk to Mio (UNDINE, designer)
